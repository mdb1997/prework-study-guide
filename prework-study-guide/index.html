HTML

  • HTML is used to create the structure of a webpage and gives framework to make it easy for user to navigate it.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • It's best to use semantic HTML elements when possible. This type of element describes its content and the role it plays on a webpage. This makes it easier to read by other developers.
  • Visible HTML element content is contained between an opening and a closing tab so that the browser understands how to treat the content.

Git

  • Git is a version control system that lets you manage and keep track of each version of your project.
  • The git command must precede every git statement and are used to save and track versions of your code.
  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git add -A: adds the starter code to the current working branch
  • git pull origin main: this insures that the local branch is in sync with the base branch in GitHub by pulling down any changes from the base branch. It's important to always to a git pull to make sure you have the latest version of the project.
  • git push origin branch name: pushes the changes from the local branch to the remote base branch on GitHub
